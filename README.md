## nuxt-blog-web

基于 nuxtjs 实现 web 端 Blog 支持 SSR

## Features

1. Package manager
   - yarn
2. Programming language

   - TypeScript

3. UI framework
   - elementUI
4. Nuxt.js module

   - axios

5. Linting tools

   - ESLint
   - Prettier
   - Lint staged files
   - StyleLint
   - CommitLint

6. Continuous integration
